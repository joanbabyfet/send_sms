## About
基于 Python3 实现批量发送短信，通过第三方服务 messagebird，解决行政人员需要批量给不同人员发送不同短信内容

## Feature

* 通过第三方服务 messagebird
* 引用 messagebird 库, 实现短信发送
* 根据给定的名单列表 list.csv 进行发送
* 按日期单独生成日志文件

## Requires
Python 3.11.0  
messagebird 2.1.0
configparser 5.3.0
pandas 1.5.2
